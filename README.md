## 프로젝트 소개
코끼리끼리(Co-KiriKiri)는 라틴어로 함께라는 뜻을 가진 ‘Co’와 여럿이 무리 짓는다는 뜻을 가진 ‘끼리끼리’를 합쳐, 함께 목표를 달성할 수 있는 스마트한 모임 플랫폼입니다.
